# TMUX - термінальний мультиплекстор

відкриваємо декілька терміналів в одному вікні
можемо створювати сесії які продовжують життя після відключення

# КОМАНДИ

  tmux new session -s назва_сесії - створення сесії та терміналу (0:zsh*)
  ctrl_b(q)+d - відєднання від сесії
  tmux attach -t назва створеної сесії - підключитись до створеної сесії

  ctrl_b(q)+c - створення додаткового термінало в поточній сесії
  ctrl_b(q)+номер_терміналу(0) - перехід в 0 термінал

  ctrl_b(q)+% - створення додаткового терміналу по вертикалі
  ctrl_b(q)+" - створення додаткового терміналу по горизонталі
  
  ctrl_b(q)+up - перехіж знизу в вверх
  ctrl_b(q)+down - перехід зверху в низ
  ctrl_b(q)+left - перехід зправа в ліво
  ctrl_b(q)+richt - перехід справа в ліво

# КОНФІГУРАЦІЯ
  
  set-option -g prefix C-q - зміна клавіши управління ctrl_b на ctrl_q
  set -g mouse on

  **конфігурація tmux проводится в конфігураційному файлі tmux.conf який може
  знаходитись в теці /etc/tmux/ або /home для користувача. Якщо даного файлу
  не існує то його потрібно створити.

  Варіант 1
  /usr/share/tmux/ команда tmux show -g > ~/.tmux.conf

  Варіант 2
  /etc/tmux.conf - якщо вам потрібна глобальна конфігурація tmux.
